
# Pedidos GS v3.5

Novedades principales:
- **Formulario de pedido** separado: arriba opciones globales y logística, abajo listado de niñas/niños.
- Campos globales: calceta niñas, zapato niñas, zapato niños, moños, pantalón, **escudos por bordar**, **fechas de entrega (múltiple)**, **entrega Ocurre/Domicilio**.
- **Datos de paquetería (foráneos)** ahora viven en **Mi perfil** de la escuela (destinatario, tel, CP, colonia, dirección, correo) y se muestran en el detalle.
- PDF actualizado con toda la información.

Migración automática: si ya existe `pedidos.db`, se agregan columnas sin perder datos.
